# AIRFORSHARE

AIRFORSHARE is a simple file sharing web application built with **Flask**, **HTML**, and **Python**.

## Project Structure
```
AIRFORSHARE/
├─ serverFiles/
│  └─ server_log.txt
├─ static/css/
│  └─ styles.css
├─ templates/
│  └─ index.html
└─ app.py
```

## Features
- Upload and share files over a local network.
- View server logs for file sharing activities.
- Simple HTML template and CSS styling.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd AIRFORSHARE
   ```

2. Install dependencies:
   ```bash
   pip install flask
   ```

3. Run the Flask app:
   ```bash
   python app.py
   ```

4. Open your browser and go to:
   ```
   http://127.0.0.1:5000
   ```

## Technologies Used
- **Flask** (Python web framework)
- **HTML/CSS** for frontend
